# Case #001: The Stolen Laptop

A browser based detective mystery game built with plain HTML, CSS, and JavaScript. You play as a detective investigating a stolen laptop, gathering evidence, interviewing suspects, and deciding who to arrest.

## How It Works

Inspect clues at the crime scene to log them on the Evidence Board. Each clue can only be collected once. Your score goes up as you find evidence and goes down a little each time you use a hint. Once all five clues are found and a suspect is selected, you can make an arrest and see whether you got it right.

## File Structure

Final Project contains an images folder with all the art, and a pages folder with index.html, style.css, and app.js.

## Running the Project

Just open pages/index.html in any modern browser. No installation needed.

## Tech Notes

Built with vanilla JavaScript, using let and const, and DOM methods like createElement instead of innerHTML.
